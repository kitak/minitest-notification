# Minitest::Notification

Test notifier for minitest via OSX Notification Center.

## Usage

    require 'minitest/autorun'
    require 'minitest/notification'

## Installation

   gem install minitest-notification 

## Dependencies

- minitest
- terminal-notifier-guard
